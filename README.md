# River Warrior Landing Page

A Nuxt landing page for a documentary film.

## Setup
1. Clone the repo:
   ```bash
   git clone https://github.com/keeganhanesboise/river-warrior-site.git
   cd river-warrior-site
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run locally:
   ```bash
   npm run dev
   ```
